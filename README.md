# Prueba Técnica - Gestor de Carpetas

**Tiempo:** 6 horas

---

## El Desafío

Implementa un gestor de carpetas jerárquico en Laravel donde:
- Los usuarios pueden crear, editar y eliminar carpetas
- Las carpetas pueden contener subcarpetas sin límite de niveles
- Cada carpeta pertenece a un usuario específico

## Preinstalación
Una vez clonado el repositorio:
- Has funcional el Laravel (environmnet, composer, ...)
- Usa MySQL/MariaDB
- Ejecuta migraciones y seeders

---

## Se evaluará:

- **Lógica y conocimiento de Laravel**
- **Buenas prácticas en código y arquitectura**
- **Validaciones y seguridad**
- **Optimización de consultas**
- **Decisiones de diseño y git**

---

## Mínimo requerido:

- Sistema CRUD funcional
- Base de datos y modelos apropiados
- Datos iniciales para testing (seeder)
- Vista navegable
- Validaciones servidor
- Borrado lógico
